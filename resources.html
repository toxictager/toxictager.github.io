Git & GitHub Basics

Version control is essential. Here are the only commands you need to start:

ONE-TIME SETUP:

git config --global user.name "Your Name"
git config --global user.email "your-email@example.com"

FOR EACH NEW PROJECT:

git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin [your-github-repo-url]
git push -u origin main

DAILY WORKFLOW:

git add .
git commit -m "Describe what you did"
git push

  • GitHub Docs - Git Basics

    Official documentation for getting started with Git. Don't overcomplicate it - basics first.

    GitHub Docs →
